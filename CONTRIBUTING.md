# Contributing to AI Compute Stack Diagram

Thank you for your interest in contributing!

## Types of Contributions

- **New diagrams**: Architecture visualizations
- **Improvements**: Clarity, accuracy
- **Documentation**: Explanations

## Diagram Guidelines

- Use ASCII art for text-based diagrams
- Keep diagrams readable in terminal
- Include legend/explanation
- Follow existing style

## Pull Request Process

1. Fork repository
2. Create feature branch
3. Preview diagrams
4. Submit PR
